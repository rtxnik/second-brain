# 11.05 Голосовые заметки

Быстрый захват идей через голосовой ввод. Особенно удобно во время прогулок, поездок, тренировок — когда набирать текст неудобно.

## Настройка через Siri Shortcuts

Создать Shortcut с голосовой командой "Заметка". Добавить действие Dictate Text, применить шаблон с timestamp, сохранить в папку inbox через Save to Files.

Workflow: голосовая команда → распознавание → шаблон → сохранение в inbox → обработка позже.

## Советы для распознавания

Чёткое произношение ключевых терминов. Короткие фразы до пары предложений. Контекст в начале: "По проекту X...". При необходимости проговаривать знаки препинания.

## Обработка

Ежедневная ревизия голосовых заметок. Категоризация по темам, связывание с существующими заметками, расширение кратких записей в полноценные.

## Альтернативы

Otter.ai для транскрипции длинных записей. Whisper API для локального распознавания. Voice Memos + ручная транскрипция если качество критично.
