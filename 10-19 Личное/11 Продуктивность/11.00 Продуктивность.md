# 11.00 Продуктивность

Методы организации информации, управления временем и эффективной работы со знаниями.

## Системы организации

[[11.01 Johnny Decimal]] — числовая иерархия для файлов и заметок, максимум 10 вариантов на каждом уровне. Хорошо работает для структурированной документации.

[[11.02 Zettelkasten]] — сеть атомарных заметок, связанных по смыслу. Подходит для исследований и накопления знаний.

[[11.03 PARA]] — четыре папки по принципу "насколько это важно сейчас". Projects, Areas, Resources, Archive.

## Работа с информацией

[[11.04 Фильтрация контента]] — как не утонуть в потоке статей, видео и подкастов.

[[11.05 Голосовые заметки]] — быстрый захват идей через Siri, пока руки заняты.

[[11.06 RSS]] — контроль над информационным потоком без алгоритмов соцсетей.

## Цели и время

[[11.07 Постановка целей]] — как формулировать цели чтобы достигать.

[[11.08 Тайм-менеджмент]] — GTD, Pomodoro, матрица Эйзенхауэра.

[[11.09 Метод Фейнмана]] — 12 задач для фильтрации информации.

## Что использую

Комбинирую Johnny Decimal для структуры папок с Zettelkasten для связей между заметками. RSS для входящего потока, голосовые заметки для захвата на ходу.
