# 34.03.10 Rocket.Chat

Open source альтернатива Slack с полным контролем над данными.

## Зачем нужен

- **Контроль данных** - переписка остаётся у тебя
- **Безопасность** - интеграция с AD/LDAP
- **Экономия** - бесплатен без ограничений
- **Кастомизация** - можно менять под нужды

## Основные фичи

- Каналы, личные сообщения, файлы
- Видео/аудио звонки
- Мобильные и десктопные приложения
- Интеграции (GitHub, Jira, мониторинг)
- Боты для автоматизации

## Сравнение

| Feature         | Rocket.Chat | Slack      | Teams       |
| --------------- | ----------- | ---------- | ----------- |
| Self-hosted     | ✅          | ❌         | Частично    |
| Бесплатно       | ✅          | Ограничено | С подпиской |
| Контроль данных | Полный      | Нет        | Частичный   |

## Развёртывание

**Docker Compose:**

```yaml
version: "3"
services:
  rocketchat:
    image: rocket.chat:latest
    environment:
      - MONGO_URL=mongodb://mongo:27017/rocketchat
```

**Требования:** 2 CPU, 4GB RAM, SSD

## Когда использовать

- Требования к контролю данных
- Slack дорог для большой команды
- Комплаенс требования
- Нужна кастомизация
