# 34.01.02 Термины Ansible

Глоссарий основных понятий Ansible.

## Архитектура

| Термин       | Определение                       |
| ------------ | --------------------------------- |
| Control Node | Управляющая машина с Ansible      |
| Managed Host | Целевая система для автоматизации |
| Inventory    | Список управляемых хостов         |
| Module       | Исполняемая единица работы        |
| Plugin       | Расширение для подключений        |
| Collection   | Набор модулей, ролей, плагинов    |

## Конфигурация

| Термин   | Определение                         |
| -------- | ----------------------------------- |
| Playbook | YAML файл с описанием автоматизации |
| Play     | Набор задач для группы хостов       |
| Task     | Атомарная операция (вызов модуля)   |
| Handler  | Задача при изменениях в системе     |
| Role     | Переиспользуемый набор задач        |
| Tag      | Метка для селективного выполнения   |

## Данные

| Термин   | Определение                             |
| -------- | --------------------------------------- |
| Variable | Именованное значение для параметризации |
| Fact     | Автоматически собираемая информация     |
| Vault    | Механизм шифрования секретов            |
| Template | Файл с переменными (Jinja2)             |

## Операции

| Термин       | Определение                         |
| ------------ | ----------------------------------- |
| Gather Facts | Автоматический сбор информации      |
| Dry Run      | Проверочный запуск (`--check`)      |
| Forks        | Количество параллельных подключений |
| Strategy     | Способ выполнения задач             |
