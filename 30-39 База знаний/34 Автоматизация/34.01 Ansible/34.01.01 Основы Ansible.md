# 34.01.01 Основы Ansible

Ansible - система автоматизации IT-инфраструктуры с открытым исходным кодом. Описывает желаемое состояние системы декларативно, а Ansible сам определяет как его достичь.

## Ключевые принципы

**Agentless** - работает без агентов, достаточно SSH доступа
**Идемпотентность** - повторное выполнение не меняет результат
**Декларативность** - описываем "что хотим", а не "как делать"
**Простота** - YAML конфигурация, минимум зависимостей

## Технологии

- YAML для описания конфигураций
- Jinja2 для шаблонизации
- SSH для подключения (без агентов)
- Python на managed hosts
