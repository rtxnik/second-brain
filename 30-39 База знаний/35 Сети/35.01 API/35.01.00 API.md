**Автоматизация:** API для управления инфраструктурой ([[34.01.00 Ansible|Ansible]], Terraform)

**CI/CD интеграция:** webhooks и API для пайплайнов

**Мониторинг:** API для сбора метрик и алертинга ([[34.02.10 Zabbix]] API)

**Инструменты:** API для интеграции DevOps инструментов ([[34.03.10 Rocket.Chat]] webhooks)
