# 31.04.50 Установка XCP-ng

Развертывание [[31.04.02 XCP-ng]] как bare-metal гипервизора на физическое оборудование.

## Требования

### Минимальные

- CPU: 64-bit x86 с Intel VT-x или AMD-V
- RAM: 2GB (рекомендуется 4GB+)
- Storage: 46GB
- Network: сетевая карта

### Рекомендуемые

- CPU: Intel Xeon / AMD EPYC
- RAM: 32GB+
- Storage: SSD для системы + отдельные диски для VM
- Network: несколько интерфейсов

## Подготовка

**1. Скачать ISO:**

```
https://xcp-ng.org/download/
```

**2. Создать загрузочный носитель:**

```bash
# Linux
sudo dd if=xcp-ng-8.2.0.iso of=/dev/sdX bs=4M status=progress

# Windows - Rufus
```

**3. Настроить BIOS:**

- Включить Intel VT-x / AMD-V
- Включить Intel VT-d / AMD IOMMU (для PCI passthrough)
- Приоритет загрузки с USB

## Процесс установки

**4. Загрузка** — выбрать "Install XCP-ng"

**5. Базовая конфигурация:**

- Клавиатура
- Принятие лицензии
- Выбор диска

**6. Сеть:**

```
IP: 192.168.1.100
Netmask: 255.255.255.0
Gateway: 192.168.1.1
DNS: 8.8.8.8
Hostname: xcp-ng-01.local
```

**7. Время:** timezone + NTP

**8. Пароль root:** минимум 8 символов

**9. Завершение:** подтвердить, дождаться, перезагрузить

## После установки

**Проверка:**

```bash
xe host-list
ip addr show
curl -k https://192.168.1.100
```

**Веб-интерфейс:** `https://192.168.1.100`

- Username: root
- Password: (установленный)

## Безопасность

- SSH ключи вместо паролей
- Обновить до последних патчей
- Настроить файрвол
- Резервные копии конфигурации
