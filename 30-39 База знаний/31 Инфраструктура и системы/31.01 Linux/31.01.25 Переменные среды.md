# 31.01.25 Переменные среды

Динамические значения для настройки окружения ОС.

## Стандартные переменные

```bash
echo $PATH      # где искать исполняемые файлы
echo $HOME      # домашний каталог
echo $USER      # текущий пользователь
echo $SHELL     # текущая оболочка
echo $PWD       # рабочий каталог
echo $EDITOR    # редактор по умолчанию
```

## Управление

```bash
printenv                    # все переменные
env                         # то же
export VAR="value"          # создать/изменить
export PATH=$PATH:/new/path # добавить к PATH
unset VAR                   # удалить
```

Изменения действуют только в текущей сессии.

## Постоянные переменные

Добавить в файлы конфигурации оболочки:

```bash
# bash
~/.bashrc          # интерактивные сессии
~/.bash_profile    # при входе

# zsh
~/.zshrc
~/.zprofile
```

Пример:

```bash
echo 'export MY_VAR="value"' >> ~/.bashrc
source ~/.bashrc
```

## Частые примеры

```bash
export PATH=$PATH:/home/user/bin
export EDITOR=vim
export JAVA_HOME=/usr/lib/jvm/java-11-openjdk
```
