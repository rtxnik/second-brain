# 31.01.51 PXE Netboot

Установка ОС по сети без физических носителей.

## Netboot

Общий термин для сетевой загрузки. Реализуется через PXE, BOOTP, TFTP, HTTP.

## PXE (Preboot Execution Environment)

Самый популярный протокол. Использует DHCP + TFTP.

### Процесс

1. DHCP Discovery - клиент запрашивает настройки
2. DHCP Offer - сервер отдаёт IP и путь к TFTP
3. TFTP Download - клиент загружает образ
4. Booting - установка ОС

## Настройка PXE сервера

1. DHCP сервер - выдача IP и пути до TFTP
2. TFTP сервер - минимальный загрузочный образ
3. NFS/HTTP/FTP сервер - установочные файлы

## Преимущества

- Масштабируемость - много машин одновременно
- Автоматизация через скрипты и конфиги
- Централизованное хранилище образов
- Нет затрат на физические носители

## Альтернативы

- HTTP Boot - загрузка через HTTP
- BOOTP - для устаревших систем
- NFS - сетевые диски (Raspberry Pi)
