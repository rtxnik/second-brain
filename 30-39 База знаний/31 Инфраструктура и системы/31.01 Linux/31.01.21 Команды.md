# 31.01.21 Базовые команды

Минимальный набор утилит для работы в терминале.

## Навигация

```bash
ls              # список файлов
ls -la          # детально со скрытыми
cd /path        # перейти
cd ..           # вверх
cd ~            # домой
pwd             # текущий путь
```

## Файлы и каталоги

```bash
mkdir dir           # создать каталог
mkdir -p a/b/c      # создать с путём
touch file          # создать файл
cp src dst          # копировать
cp -r src dst       # копировать каталог
mv src dst          # переместить/переименовать
rm file             # удалить
rm -rf dir          # удалить каталог (осторожно!)
```

## Просмотр файлов

```bash
cat file            # вывести содержимое
less file           # постраничный просмотр
head -n 20 file     # первые 20 строк
tail -f file        # следить за файлом (логи)
```

## Поиск

```bash
grep "text" file    # искать текст
grep -r "text" dir  # рекурсивно
find /path -name "*.log"
```

## Процессы

```bash
ps aux              # все процессы
kill PID            # завершить
kill -9 PID         # принудительно
```

## Сеть и диски

```bash
ping host           # проверить соединение
ip addr             # сетевые интерфейсы
ss -tuln            # открытые порты
df -h               # место на дисках
du -sh dir          # размер каталога
```

## Справка

```bash
man command         # полная документация
command --help      # краткая справка
```
