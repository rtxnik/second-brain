# 31.01.18 Командные оболочки

Shell - интерфейс для взаимодействия с ОС. Если не уверен - бери bash.

## sh (Bourne Shell)

1979, POSIX стандарт. Минимальный набор функций. Для скриптов, совместимых везде. Есть даже в роутерах.

## bash (Bourne Again Shell)

1989, стандарт для Linux. Всё от sh + история команд, автодополнение, массивы. Файлы: .bashrc, .bash_profile
Рекомендую: скрипты на bash работают одинаково локально и на сервере.

## zsh (Z Shell)

1990, расширенные возможности. Плагины через Oh-My-Zsh, улучшенное автодополнение, глобальные алиасы.
Стандарт в macOS.

## fish

2005, акцент на удобстве. Контекстное автодополнение, веб-интерфейс настройки. Собственный синтаксис, не POSIX.

## Выбор

sh - максимальная совместимость
bash - универсальный выбор
zsh - для кастомизации
fish - для интерактивной работы
