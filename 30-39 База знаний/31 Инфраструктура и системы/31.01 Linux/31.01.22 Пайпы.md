# 31.01.22 Пайпы и перенаправления

Три стандартных потока: stdin (0), stdout (1), stderr (2).

## Перенаправление вывода

```bash
command > file      # записать (перезаписать)
command >> file     # дописать в конец
command 2> file     # только ошибки
command > file 2>&1 # всё в один файл
```

## Перенаправление ввода

```bash
command < file      # файл как ввод
wc -l < input.txt   # посчитать строки
```

## Пайпы (конвейеры)

Вывод одной команды в ввод другой:

```bash
cat log.txt | grep "error"
ps aux | grep nginx | head -5
cat access.log | grep "404" | awk '{print $1}' | sort | uniq -c
```

## Примеры

```bash
# сохранить вывод
df -h > disk_usage.txt

# ошибки в файл
find / -name "*.log" 2> errors.txt

# вывод и ошибки + на экран
(ls -l /invalid 2>&1) | tee output.txt
```

Размер конвейера не ограничен.
