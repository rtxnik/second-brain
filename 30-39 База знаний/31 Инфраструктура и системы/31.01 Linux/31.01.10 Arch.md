# 31.01.10 Arch Linux

Rolling release дистрибутив по принципу KISS (Keep It Simple, Stupid). Минимальная база, пользователь настраивает с нуля.

## Философия

Простота, элегантность, контроль. Нет графического инсталлятора - установка через CLI с полным пониманием процесса.

## Пакетный менеджер

Pacman - быстрый и простой. Rolling release = постоянные обновления без мажорных релизов. AUR (Arch User Repository) - пакеты от сообщества.

## Arch Wiki

Лучшая документация в Linux мире. Полезна даже для других дистрибутивов.

## Производные

Manjaro (удобный инсталлятор), EndeavourOS (Arch с GUI установкой), ArcoLinux (обучающий).
Arch учит понимать Linux "изнутри".
