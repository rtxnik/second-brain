# 31.01.70 Чит-лист

Краткий справочник команд.

## Навигация

```bash
ls -la           # список с правами
cd /path         # перейти
pwd              # текущий путь
find /path -name "*.log"
which command    # расположение команды
```

## Файлы

```bash
cp src dst       # копировать
mv old new       # переместить
rm -rf dir       # удалить
chmod 755 file   # права
chown user:group file
ln -s target link
```

## Просмотр

```bash
cat file         # вывести
less file        # постранично
head -20 file    # первые 20 строк
tail -f file     # следить
grep -r "text" dir
```

## Процессы

```bash
ps aux           # список
top / htop       # мониторинг
kill -9 PID      # завершить
nohup cmd &      # в фоне
```

## Сеть

```bash
curl -I url      # заголовки
wget url         # скачать
ssh user@host
scp file user@host:
netstat -tulpn   # порты
```

## Архивы

```bash
tar -czf arch.tar.gz dir/
tar -xzf arch.tar.gz
zip -r arch.zip dir/
unzip arch.zip
```

## Права (chmod)

```
4=r  2=w  1=x
755 = rwxr-xr-x (каталоги)
644 = rw-r--r-- (файлы)
```
