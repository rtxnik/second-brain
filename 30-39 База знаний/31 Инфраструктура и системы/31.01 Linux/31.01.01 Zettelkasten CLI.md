# 31.01.01 Zettelkasten CLI

Инструмент командной строки для создания и управления заметками. Написан на Python Мишей Ван Дер Бергом. Устраняет ручное создание файлов и добавление метаданных.

## Установка

```bash
python3 --version
pipx --version

pipx install git+https://github.com/mischavandenburg/zettelkasten-cli.git
```

## Основные команды

```bash
# Новая заметка
zettelkasten new "Название заметки"

# Открыть ежедневную заметку
zettelkasten day

# Справка
zettelkasten --help
```

## Рабочий процесс

Создание заметки через CLI → автоматическое связывание с daily notes → переход в Obsidian для доработки → связывание с другими заметками.

## Зачем это нужно

Экономия времени на создании файлов. Консистентность в именовании. Интеграция с daily notes workflow. Работа из любого терминала без открытия Obsidian.
