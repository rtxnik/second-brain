# 31.01.71 Горячие клавиши терминала

Ускоряют работу в CLI.

## Навигация по строке

```
Ctrl+A    начало строки
Ctrl+E    конец строки
Alt+F     слово вперёд
Alt+B     слово назад
```

## Редактирование

```
Ctrl+U    удалить до начала
Ctrl+K    удалить до конца
Ctrl+W    удалить слово назад
Alt+D     удалить слово вперёд
Ctrl+Y    вставить удалённое
```

## История

```
Ctrl+R    поиск по истории
Ctrl+P    предыдущая команда
Ctrl+N    следующая команда
!!        повторить последнюю
!n        команда номер n
```

## Управление

```
Ctrl+C    прервать (SIGINT)
Ctrl+Z    в фон (SIGTSTP)
Ctrl+D    EOF / выход
Ctrl+L    очистить экран
```

## Автодополнение

```
Tab       дополнить
Tab Tab   все варианты
```

## Буфер

```
Ctrl+Shift+C    копировать
Ctrl+Shift+V    вставить
```
