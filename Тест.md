# 11.01 Zettelkasten

Метод ведения заметок через сеть связанных идей. Название от немецкого "ящик с карточками" - так Никлас Луман вёл свою картотеку.
Главная идея - одна заметка содержит одну мысль, которая понятна сама по себе. Не папки и категории, а связи по смыслу. Получается что-то вроде второго мозга, где идеи цепляются друг за друга. 

## Как это работает

Есть несколько типов заметок. Fleeting notes — это быстрые записи на ходу, их нужно разобрать в тот же день. Literature notes — конспекты из книг и статей, обязательно своими словами. Permanent notes — уже обработанные мысли, которые останутся надолго.
Процесс простой: записал идею, переформулировал своими словами, нашёл с чем связать из существующих заметок, добавил ссылки.

## Почему это работает

В отличие от папок, где файл лежит в одном месте, тут заметка может быть связана с чем угодно. Через месяц открываешь старую заметку и видишь связи, о которых уже забыл.
