# 32.01.06 Daemon

Docker Daemon (`dockerd`) - это фоновый сервис, который управляет всеми объектами Docker. **Демон обрабатывает команды от клиента через REST API и выполняет все операции с контейнерами, образами, сетями и томами**.
Демон напрямую взаимодействует с ядром Linux через [[31.01.16 Namespaces]] и [[31.01.17 Cgroups]] для создания изолированных сред выполнения. Управляет полным жизненным циклом контейнеров - создание, запуск, остановка, удаление.
Демон также отвечает за сборку образов из Dockerfile, управление локальным кешем образов и взаимодействие с реестрами для загрузки и отправки образов.
Конфигурация демона задается файлом `/etc/docker/daemon.json` или параметрами командной строки. Основные настройки включают директорию данных, драйвер логирования, сетевые настройки и ограничения ресурсов.
В настоящее время демон работает только на Linux, так как зависит от функций ядра Linux. На Windows и macOS Docker использует виртуальную машину Linux для запуска демона.
