# 32.01.05 Client

Docker Client — это интерфейс командной строки для взаимодействия с демоном Docker. **Клиент переводит пользовательские команды в HTTP запросы к REST API демона**.

Основные команды включают `docker run` (запуск контейнера), `docker build` (сборка образа), `docker pull/push` (работа с реестрами), `docker ps` (список контейнеров), `docker images` (список образов).

Клиент может работать локально с демоном на той же машине или подключаться к удаленному демону через TCP/SSH. Настройки подключения задаются переменными среды `DOCKER_HOST`, `DOCKER_TLS_VERIFY`.

Docker CLI поддерживает автодополнение команд и контекстную справку через `docker --help` и `docker <command> --help`. Это значительно упрощает освоение инструмента.

Клиент также поддерживает конфигурационные файлы для хранения настроек подключения к различным демонам и реестрам.
