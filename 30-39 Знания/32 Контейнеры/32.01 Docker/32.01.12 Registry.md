# 32.01.12 Registry

Docker Registry — это сервис для хранения и распространения образов Docker. **Registry позволяет командам совместно использовать образы и автоматизировать процессы деплоймента**.

## Docker Hub

Публичный облачный реестр по умолчанию. Содержит официальные образы популярных приложений (`nginx`, `postgres`, `node`) и миллионы образов от сообщества. Поддерживает автоматическую сборку образов из Git репозиториев.

## Приватные реестры

| Решение | Особенности |
|---------|-------------|
| Harbor | Enterprise, RBAC, сканирование |
| AWS ECR | Интеграция с AWS |
| Google GCR | Google Cloud |
| Azure ACR | Microsoft Azure |
| GitLab CR | Интеграция с GitLab CI |

## Команды

```bash
docker pull nginx:latest          # загрузка образа
docker push myapp:v1              # отправка образа
docker search nginx               # поиск в Docker Hub
docker login registry.company.com # аутентификация
```

## Именование

```
registry.company.com/team/app:v1.0
```

Если реестр не указан явно, используется Docker Hub (`docker.io` по умолчанию).

## Локальный реестр

```bash
docker run -d -p 5000:5000 registry:2
docker push localhost:5000/myapp:v1
```

Для приватных реестров требуется аутентификация через `docker login`. Credential helper может интегрироваться с корпоративными системами управления секретами.
