# 34.02.10 Zabbix

Enterprise-класс система мониторинга с открытым исходным кодом для отслеживания IT-инфраструктуры.

## Архитектура

| Компонент | Функция |
|-----------|---------|
| Server | Сбор, обработка, анализ данных |
| Agent | Сбор метрик на хостах |
| Proxy | Промежуточный сборщик |
| Web UI | Управление и визуализация |
| Database | Хранение конфигурации и данных |

## Методы сбора данных

- **Zabbix Agent** — активный/пассивный режим
- **SNMP** — сетевое оборудование
- **IPMI** — аппаратный мониторинг
- **JMX** — Java-приложения
- **HTTP checks** — веб-сервисы

## Возможности

**Мониторинг:**
- CPU, RAM, диск, сеть
- Сервисы (HTTP, SSH, DNS)
- Логи приложений
- Базы данных

**Алертинг:**
- Триггеры с условиями
- Многоуровневая эскалация
- Email, SMS, Telegram, Slack

**Визуализация:**
- Настраиваемые dashboards
- Графики и тренды
- Топологические карты

## Безопасность

- PSK шифрование
- Ролевой доступ
- HTTPS для веб-интерфейса
- API аутентификация
