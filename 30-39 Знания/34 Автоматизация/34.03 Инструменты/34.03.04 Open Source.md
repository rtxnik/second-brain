# 34.03.04 Open Source

Модель разработки ПО с открытым исходным кодом, правом модификации и распространения.

## Принципы

- **Прозрачность** - открытый код для всех
- **Коллаборация** - коллективная разработка
- **Свобода** - модификация под свои нужды
- **Сообщество** - улучшения доступны всем

## Преимущества

| Аспект         | Open Source | Проприетарное |
| -------------- | ----------- | ------------- |
| Прозрачность   | Полная      | Закрытый код  |
| Стоимость      | Бесплатно   | Лицензии      |
| Кастомизация   | Полная      | Ограниченная  |
| Vendor Lock-in | Нет         | Да            |
| Безопасность   | Аудит кода  | Доверие       |

## Лицензии

| Лицензия   | Особенности                          |
| ---------- | ------------------------------------ |
| GPL        | Copyleft, производные работы открыты |
| MIT        | Максимальная свобода                 |
| Apache 2.0 | Защита от патентов                   |
| BSD        | Минимальные ограничения              |

## Бизнес-модели

- **Support & Services** - Red Hat, Canonical
- **Open Core** - базовая версия + enterprise
- **SaaS** - управляемые сервисы
- **Dual Licensing** - открытая + коммерческая
