# 34.03.30 Установка Rocket.Chat

Установка [[34.03.10 Rocket.Chat]] через Docker Compose с SSL.

## Требования

- **CPU:** 2 cores
- **RAM:** 4GB (рекомендуется 8GB)
- **Storage:** 20GB SSD
- **Docker:** последняя версия

## Подготовка

```bash
# Структура проекта
mkdir ~/rocket-chat && cd ~/rocket-chat
mkdir -p data/db data/uploads
```

## docker-compose.yml

```yaml
version: '3.8'

services:
  rocketchat:
    image: registry.rocket.chat/rocketchat/rocket.chat:latest
    restart: unless-stopped
    environment:
      MONGO_URL: mongodb://mongo:27017/rocketchat
      ROOT_URL: https://chat.example.com
      PORT: 3000
    depends_on:
      - mongo
    ports:
      - "3000:3000"
    volumes:
      - ./data/uploads:/app/uploads

  mongo:
    image: mongo:4.4
    restart: unless-stopped
    volumes:
      - ./data/db:/data/db
    command: mongod --oplogSize 128 --replSet rs0
```

## Запуск

```bash
docker-compose up -d
docker-compose logs -f rocketchat
```

## Настройка

1. Открыть https://chat.example.com
2. Пройти мастер настройки
3. Создать администратора
4. Настроить SMTP для email

## Backup

```bash
# MongoDB dump
docker-compose exec mongo mongodump --out /dump/backup

# Uploads
cp -r data/uploads backup/
```
