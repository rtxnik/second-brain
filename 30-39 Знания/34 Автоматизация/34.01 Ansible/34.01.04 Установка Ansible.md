# 34.01.04 Установка Ansible

Ansible устанавливается только на control node. Managed hosts требуют лишь SSH и Python.

## Требования

**Control Node:** Linux/macOS/WSL2, Python 3.8+, SSH клиент
**Managed Hosts:** SSH сервер, Python 2.7+/3.5+, sudo доступ

## Установка

**Ubuntu/Debian:**

```bash
sudo apt update
sudo apt install ansible
```

**Через pip (рекомендуется):**

```bash
python3 -m venv ansible-env
source ansible-env/bin/activate
pip install ansible
```

**macOS:**

```bash
brew install ansible
```

## Настройка SSH

```bash
ssh-keygen -t ed25519 -C "ansible@control-node"
ssh-copy-id -i ~/.ssh/id_ed25519.pub user@managed-host
```

## Базовая конфигурация

```ini
# ansible.cfg
[defaults]
inventory = ./inventory.ini
remote_user = ansible
private_key_file = ~/.ssh/id_ed25519
host_key_checking = False

[ssh_connection]
pipelining = True
```

## Проверка

```bash
ansible --version
ansible all -m ping
```
