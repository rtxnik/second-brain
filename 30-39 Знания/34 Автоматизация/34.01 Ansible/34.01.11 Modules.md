# 34.01.11 Modules

Ansible Modules — исполняемые единицы работы на managed hosts.

## Основные категории

**Управление пакетами:**
- `ansible.builtin.apt` — Debian/Ubuntu
- `ansible.builtin.yum/dnf` — RedHat/CentOS
- `ansible.builtin.package` — универсальный

**Работа с файлами:**
- `ansible.builtin.copy` — копирование файлов
- `ansible.builtin.file` — управление файлами
- `ansible.builtin.template` — генерация из шаблонов

**Управление сервисами:**
- `ansible.builtin.service` — запуск/остановка
- `ansible.builtin.systemd` — расширенное управление

## Примеры

```yaml
- name: "Install nginx"
  ansible.builtin.apt:
    name: nginx
    state: present
    update_cache: true

- name: "Start nginx"
  ansible.builtin.service:
    name: nginx
    state: started
    enabled: true
```

## Collections

Модули организованы в коллекции:
- `ansible.builtin.*` — встроенные
- `community.general.*` — сообщество
- `amazon.aws.*` — AWS специфичные
