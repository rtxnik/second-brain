# 34.01.18 Galaxy

Ansible Galaxy — централизованный репозиторий ролей и коллекций.

## Поиск и установка

```bash
# Поиск ролей
ansible-galaxy search nginx

# Установка роли
ansible-galaxy install geerlingguy.nginx

# Установка версии
ansible-galaxy install geerlingguy.mysql,1.4.0

# Установка из requirements
ansible-galaxy install -r requirements.yml
```

## requirements.yml

```yaml
roles:
  - name: geerlingguy.nginx
    version: 2.8.0
  - name: geerlingguy.mysql
    
collections:
  - name: community.general
    version: ">=1.0.0"
  - name: ansible.posix
```

## Управление ролями

```bash
ansible-galaxy list              # установленные
ansible-galaxy info role_name    # информация
ansible-galaxy remove role_name  # удаление
```

## Использование

```yaml
- hosts: webservers
  roles:
    - role: geerlingguy.nginx
      vars:
        nginx_vhosts:
          - listen: "80"
            server_name: "example.com"
```

## Коллекции

```bash
ansible-galaxy collection install community.general
ansible-galaxy collection list
```
