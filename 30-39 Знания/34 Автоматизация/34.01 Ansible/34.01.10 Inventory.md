# 34.01.10 Inventory

Ansible Inventory определяет список управляемых хостов и их параметры.

## Статический inventory

**INI формат:**
```ini
[webservers]
web1.example.com
web2.example.com

[databases]
db1.example.com ansible_host=192.168.1.20

[production:children]
webservers
databases
```

**YAML формат:**
```yaml
all:
  children:
    webservers:
      hosts:
        web1.example.com:
        web2.example.com:
    databases:
      hosts:
        db1.example.com:
          mysql_port: 3306
```

## Переменные хостов

```ini
[webservers]
web1 ansible_host=192.168.1.10 ansible_user=ubuntu

[webservers:vars]
http_port=80
```

## Динамический inventory

Генерируется из облачных провайдеров (AWS, GCP, Azure) через плагины.

## Команды

```bash
ansible-inventory --list          # все хосты
ansible-inventory --host hostname # переменные хоста
ansible all --list-hosts          # список для группы
```
