# 34.01.12 Plugins

Ansible Plugins расширяют функциональность и работают на control node.

## Типы plugins

**Connection** - управление подключением:

- `ssh` - стандартное SSH
- `local` - выполнение на control node
- `docker` - подключение к контейнерам

**Inventory** - динамическое получение хостов:

- `aws_ec2` - AWS EC2 инстансы
- `gcp_compute` - Google Cloud
- `azure_rm` - Azure VM

**Lookup** - получение данных:

- `file` - чтение локальных файлов
- `env` - переменные окружения
- `password` - генерация паролей

## Примеры lookup

```yaml
- name: "Read file content"
  ansible.builtin.debug:
    msg: "{{ lookup('file', '/etc/hostname') }}"

- name: "Get environment variable"
  ansible.builtin.debug:
    msg: "{{ lookup('env', 'HOME') }}"
```

## Filter plugins

```yaml
# Обработка строк
msg: "{{ 'Hello' | lower | replace(' ', '_') }}"

# JSON/YAML
config_data: "{{ json_string | from_json }}"
```
