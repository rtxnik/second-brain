# 31.04.03 Xen Orchestra

Xen Orchestra (XO) — веб-интерфейс для управления [[31.04.02 XCP-ng]] и XenServer. Централизованное управление виртуальной инфраструктурой через браузер.

Разворачивается как VM на управляемом гипервизоре.

## Возможности

**Управление VM** — создание, настройка, миграция, удаление

**Мониторинг** — real-time статистика хостов и VM

**Резервное копирование** — автоматические backup с дедупликацией

**Сети** — настройка виртуальных сетей, VLAN, SR-IOV

**Хранилища** — подключение и управление storage

## Архитектура

| Компонент | Технология |
|-----------|------------|
| Web UI | React, responsive design |
| REST API | Полный программный доступ |
| CLI | Автоматизация через командную строку |
| Плагины | Расширение функциональности |

## Интеграции

Ansible — автоматизация развертывания VM

Terraform — Infrastructure as Code

API — собственные скрипты и приложения

## Лицензирование

| Версия | Особенности |
|--------|-------------|
| XOA (Appliance) | Коммерческая, поддержка, готовый образ |
| XO from Sources | Бесплатная, полный функционал, ручная сборка |
| XO Community | Готовые сборки сообщества |

## Сравнение

vs XenCenter — современный веб-интерфейс, доступ с любого устройства

vs vCenter — легче в развертывании, не требует лицензирования
