# 31.04.02 XCP-ng

XCP-ng — открытый гипервизор типа 1 (bare-metal) на базе Citrix XenServer. Enterprise-класс виртуализации без лицензионных ограничений.

Устанавливается на физическое оборудование, обеспечивает прямой доступ к аппаратным ресурсам.

## Характеристики

**Производительность** — минимальные накладные расходы через Xen паравиртуализацию

**Масштабируемость** — до 64 хостов в кластере, тысячи VM

**Безопасность** — изоляция на уровне гипервизора

**Совместимость** — экосистема XenServer без vendor lock-in

## Функции Enterprise

| Функция | Описание |
|---------|----------|
| Live Migration | Миграция работающих VM без простоя |
| High Availability | Автоперезапуск VM при сбоях хоста |
| Storage Replication | Репликация между хранилищами |
| Snapshots | Моментальные снимки состояния |
| Backup/Restore | Резервное копирование |

## Управление

**[[31.04.03 Xen Orchestra]]** — веб-интерфейс с REST API (основной)

**xe-cli** — командная строка для автоматизации

**XenCenter** — Windows-клиент (совместимость)

**API** — интеграция с Terraform, Ansible

## Сценарии использования

Серверная виртуализация — консолидация нагрузок

VDI — виртуальные рабочие столы

Частные облака — основа облачной платформы

Homelab — изучение и тестирование

## Отличия от конкурентов

| vs VMware | vs Hyper-V |
|-----------|------------|
| Все функции бесплатно | Работает на любом железе |
| Открытый исходный код | Не привязан к Microsoft |
