# 31.01.28 LXD

Современный менеджер поверх LXC. Превращает низкоуровневые возможности в enterprise-ready платформу с REST API.

## Возможности

- Системные контейнеры (LXC) и VM (QEMU) через единый интерфейс
- Снимки (snapshots) с ZFS/Btrfs
- Live миграция между хостами
- Backup/restore
- Профили конфигурации
- Кластерный режим

## REST API

Все операции через HTTP API - легко интегрировать с CI/CD и автоматизацией.

## Сеть

Bridge, macvlan, SR-IOV. Встроенный DHCP и DNS.

## Команды

```bash
lxc launch ubuntu:22.04 mycontainer
lxc list
lxc exec mycontainer -- bash
lxc snapshot mycontainer snap1
lxc stop mycontainer
```

## Сценарии

Хостинг провайдеры, dev environments, edge computing - где нужна изоляция на уровне ОС с возможностями enterprise виртуализации.
