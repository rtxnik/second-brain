# 31.01.04 Chezmoi

Инструмент управления dotfiles с поддержкой шаблонов и секретов. Решает проблему синхронизации конфигураций между разными машинами и ОС. Написан на Go.

## Возможности

Шаблонизация - адаптация под разные машины. Управление секретами - безопасное хранение паролей. Машинно-специфичные настройки. Сухие прогоны перед применением.

## Основные команды

```bash
# Инициализация
chezmoi init

# Добавить файл под управление
chezmoi add ~/.bashrc

# Применить изменения
chezmoi apply

# Посмотреть различия
chezmoi diff

# Редактировать
chezmoi edit ~/.bashrc
```

## Шаблоны

```bash
{{- if eq .chezmoi.os "linux" }}
editor: vim
{{- else if eq .chezmoi.os "darwin" }}
editor: code
{{- end }}
```

## Секреты

Интеграция с 1Password CLI, Bitwarden CLI, Pass, Keychain на macOS. Секреты не хранятся в репозитории в открытом виде.

## Преимущества

По сравнению с обычными [[31.01.02 Dotfiles]] - адаптация под разные системы, безопасное хранение чувствительных данных, проверка изменений перед применением.
