# 31.01.02 Dotfiles

Конфигурационные файлы Unix-систем, начинающиеся с точки. Настройки оболочки, редакторов, терминалов, инструментов разработки. Версионирование через Git обеспечивает консистентность между машинами.

## Основные файлы

**Shell:** `.bashrc`, `.zshrc` - настройки оболочки. `.profile` - переменные среды. `.aliases` - сокращения команд.
**Инструменты:** `.gitconfig` - настройки Git. `.vimrc`, `.tmux.conf` - редакторы. `.ssh/config` - SSH подключения.

## Управление

Git репозиторий для версионирования. Символические ссылки из репозитория в домашнюю директорию. Скрипты установки для развёртывания на новых машинах.

## Структура репозитория

```
dotfiles/
├── .bashrc
├── .gitconfig
├── .vimrc
├── install.sh
└── README.md
```

## Инструменты

**GNU Stow** - простое управление через симлинки.
