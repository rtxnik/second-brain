# 31.01.20 Алиасы

Короткие команды для длинных. Ускоряют работу и снижают ошибки.

## Создание

```bash
alias ll='ls -la'              # временный (до конца сессии)
# добавить в ~/.bashrc         # постоянный
```

## Популярные алиасы

```bash
alias ll='ls -la'
alias la='ls -A'
alias grep='grep --color=auto'
alias ..='cd ..'
alias ...='cd ../..'
alias ports='netstat -tuln | grep LISTEN'
```

## Управление

```bash
alias                # показать все
unalias name         # удалить
\ls                  # выполнить оригинальную команду (игнорируя алиас)
```

## Расположение

Пользовательские: ~/.bashrc, ~/.zshrc
Системные: /etc/profile

Пользовательские имеют приоритет.
