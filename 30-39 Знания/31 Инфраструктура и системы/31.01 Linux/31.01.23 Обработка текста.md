# 31.01.23 Обработка текста

Три мощных инструмента: grep, sed, awk.

## grep - поиск

```bash
grep "error" file           # найти строки с "error"
grep -i "error" file        # без учёта регистра
grep -v "debug" file        # исключить
grep -r "pattern" dir       # рекурсивно
grep -E "err|warn" file     # несколько паттернов
```

## sed - потоковое редактирование

```bash
sed 's/old/new/g' file      # замена всех вхождений
sed 's/old/new/' file       # только первое в строке
sed '1,5d' file             # удалить строки 1-5
sed -n '10,20p' file        # вывести строки 10-20
sed -i 's/old/new/g' file   # изменить файл на месте
```

## awk - обработка по полям

```bash
awk '{print $1, $3}' file           # первое и третье поле
awk '$3 > 100 {print $1}' file      # условие
awk -F: '{print $1}' /etc/passwd    # свой разделитель
awk '{sum+=$1} END {print sum}'     # сумма
```

## Комбинирование

```bash
cat access.log | grep "404" | awk '{print $1}' | sort | uniq -c | sort -rn | head
```

## Регулярные выражения

. любой символ, \* ноль или более, ^ начало строки, $ конец строки, [] класс символов.
