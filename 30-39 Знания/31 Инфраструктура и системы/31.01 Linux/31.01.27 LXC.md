# 31.01.27 LXC

Linux Containers - контейнеризация на уровне ОС. Запускает полноценные Linux системы с собственным init, а не отдельные приложения (как Docker).

## Технологии

Построен на [[31.01.16 Namespaces]] (изоляция) и [[31.01.17 Cgroups]] (лимиты ресурсов). Но подход другой - LXC эмулирует традиционную VM.

## Особенности

- Полная ОС со всеми сервисами
- Внутри работает systemd
- Можно подключаться через SSH
- Стандартные пакетные менеджеры
- Unprivileged контейнеры для безопасности

## Команды

```bash
lxc-create -n mycontainer -t ubuntu
lxc-start -n mycontainer
lxc-attach -n mycontainer
lxc-ls --fancy
lxc-stop -n mycontainer
lxc-destroy -n mycontainer
```

## Сценарии

- Изоляция рабочих нагрузок
- Тестовые среды
- Замена VM где нужна полная ОС с минимальными накладными

LXC предшествовал Docker и был его основой в ранних версиях.
