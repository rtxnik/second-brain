# 31.01.50 Методы установки

Обзор подходов к установке Linux.

## Классическая

С физических/виртуальных носителей через инсталляторы. Ubuntu, Fedora - графические мастера. Arch - ручная через CLI.

## Автоматизированная

[[31.01.26 Cloud-init]] - облачные экземпляры.
Preseed (Debian/Ubuntu), Kickstart (Red Hat) - полностью автоматические установки.

## Сетевая

[[31.01.51 PXE Netboot]] - загрузка по сети для массовых развёртываний в корпоративных средах.

## Контейнерные образы

Готовые образы для Docker, Podman. Мгновенное развёртывание без установки.

## Виртуализация

Готовые VM образы для VMware, VirtualBox, KVM. Облачные AMI и шаблоны.
