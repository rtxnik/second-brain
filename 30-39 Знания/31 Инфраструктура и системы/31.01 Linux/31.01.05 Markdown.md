# 31.01.05 Markdown

Облегчённый язык разметки для форматирования текста. Создан Джоном Грубером в 2004 году. Поддерживается почти везде — от GitHub до Obsidian.

## Базовый синтаксис

```markdown
# Заголовок 1
## Заголовок 2
### Заголовок 3

**жирный** и *курсив*
~~зачёркнутый~~
`код в строке`

- список
- ещё пункт
  - вложенный

1. нумерованный
2. список

[ссылка](http://example.com)
![картинка](image.png)
```

## Блочные элементы

```markdown
> цитата

​```python
def hello():
    print("Hello")
​```

| колонка | колонка |
|---------|---------|
| данные  | данные  |
```

## Расширения

**GitHub Flavored Markdown** — чекбоксы, таблицы, подсветка кода.

**Obsidian** — wikilinks `[[]]`, теги `#тег`, математика LaTeX.

**CommonMark** — стандартизированная спецификация.

## Где используется

README в репозиториях, документация, системы заметок, статические генераторы сайтов, техническое письмо.
