# 31.01.19 Bash скрипты

Автоматизация повторяющихся задач. Текстовый файл с последовательностью команд.

## Основы

```bash
#!/bin/bash          # shebang - указывает интерпретатор
chmod +x script.sh   # права на выполнение
./script.sh          # запуск
```

## Переменные

```bash
NAME="value"         # без пробелов вокруг =
echo $NAME           # использование
echo ${NAME}         # или так
```

Аргументы: $1, $2... Количество: $#

## Условия и циклы

```bash
if [ условие ]; then
    ...
fi

for item in list; do
    ...
done
```

## Лучшие практики

set -e - выход при ошибке
Кавычки для переменных
Валидация входных данных
Информативные сообщения об ошибках
