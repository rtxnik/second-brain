# 31.01.11 Debian Ubuntu

Самое популярное семейство. Пакеты .deb, менеджер apt.

## Debian

Консервативный подход к стабильности. Цикл 2-3 года с тщательным тестированием.

Ветки: Stable (продакшн), Testing (кандидат), Unstable/Sid (bleeding edge).

## Ubuntu

Основан на Debian. Релизы каждые 6 месяцев (год.месяц - 22.04, 24.04).

LTS каждые 2 года с 5-летней поддержкой. Доминирует в облаках - AWS, GCP, Azure предоставляют как основной образ.

## apt

```
apt update          # обновить списки
apt upgrade         # обновить пакеты
apt install pkg     # установить
apt search pattern  # поиск
```

## Производные

Mint, Elementary OS, Pop!_OS, Kali Linux.
