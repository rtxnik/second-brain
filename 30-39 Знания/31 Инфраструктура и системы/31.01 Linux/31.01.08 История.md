# 31.01.08 История Linux

1991 год - Линус Торвальдс, студент из Хельсинки, создал ядро как хобби-проект для изучения Intel 386.

Название от Linus + Unix. Изначально "Freax", но админ FTP переименовал в Linux.

## GNU + Linux

Проект GNU (Richard Stallman, 1983) к 1991 году создал почти полную ОС без ядра. Ядро Linux идеально дополнило инструменты GNU = GNU/Linux.

## Философия

Принципы Unix: простота, модульность, переносимость. "Делать одну вещь хорошо" - программы комбинируются через пайпы.

Open Source - прозрачность, безопасность, адаптация. Лицензия GPL гарантирует что код остаётся свободным.

"Release early, release often" - тысячи разработчиков по всему миру.
