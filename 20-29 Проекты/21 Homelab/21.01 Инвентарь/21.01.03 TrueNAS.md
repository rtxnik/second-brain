# 21.01.03 TrueNAS

> NAS сервер для бэкапов и файлового хранилища

## Спецификация

| Компонент | Значение                                    |
| --------- | ------------------------------------------- |
| CPU       | AMD Ryzen 5 5600G                           |
| RAM       | 64GB DDR4                                   |
| OS Disk   | NVMe 256GB                                  |
| HBA       | LSI 9300-8i (SAS3008, IT mode)              |
| Cage      | Icy Dock ExpressCage MB038SP-B (8-bay 2.5") |
| OS        | TrueNAS Scale                               |
