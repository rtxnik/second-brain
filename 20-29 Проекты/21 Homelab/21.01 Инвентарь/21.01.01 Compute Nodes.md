## Сетевые интерфейсы

### eth0 (Intel встроенный)

```
Подключение: MikroTik (trunk port)
Bridge:      vmbr0 (VLAN-aware)
VLANs:       10, 20, 30, 40, 50
Назначение:  Management + VM networking
```

### eth1 (TP-Link USB)

```
Подключение: MikroTik (access port, untagged)
Bridge:      vmbr1
Сеть:        10.0.11.0/24
Назначение:  Corosync + Ceph traffic
```

---

## Disk Layout

```
/dev/nvme0n1 (2TB Samsung 990 Pro)
├── Part 1: EFI (512MB)
├── Part 2: Proxmox OS (~100GB)
└── Part 3: Ceph OSD (~1.9TB)

/dev/nvme1n1 (2TB Samsung 990 Pro)
└── Full disk: Ceph OSD (~2TB)
```
