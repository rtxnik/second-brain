# 21.01.04 Питание и стойка

> UPS и серверная стойка

## APC Back-UPS BX1600MI

| Параметр  | Значение                 |
| --------- | ------------------------ |
| Мощность  | 1600VA / 900W            |
| Тип       | Line-Interactive         |
| Выходы    | 6x IEC C13               |
| Runtime   | ~28 мин при 19% нагрузке |
| Интерфейс | USB (для NUT)            |

### NUT конфигурация

```
Master:     TrueNAS (10.0.30.10)
Secondary:  pve-compute-01, 02, 03
Порог:      50% заряда → shutdown
```
