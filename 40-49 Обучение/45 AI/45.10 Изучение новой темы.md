# 45.10 Изучение новой темы

Промпт для структурированного изучения технологии с AI и создания заметок Zettelkasten.

## Промпт

```
Я хочу изучить технологию [название]. Предоставь информацию:

1. Базовое определение и назначение технологии
2. История развития и ключевые этапы эволюции
3. Основные принципы работы и фундаментальные концепции
4. Актуальные варианты реализации и применения
5. Преимущества и сильные стороны по сравнению с альтернативами
6. Ограничения, недостатки и потенциальные проблемы
7. Необходимые технические знания и навыки для освоения
8. Рекомендуемые ресурсы (книги, курсы, документация)
9. Примеры успешного внедрения в реальных проектах
10. Перспективы развития и актуальные тренды

Для создания заметок Zettelkasten в Obsidian:

11. Разбей на модульные концепции для порционного изучения
12. Для каждого модуля предложи структуру заметки:
    - Краткое определение (1-2 предложения)
    - Ключевые моменты для фиксации
    - Формат представления (текст, диаграмма, код, таблица)
    - Рекомендуемые связи с другими концепциями
13. Предложи иерархию связей и порядок изучения
14. Укажи точки расширения для связей с другими технологиями
15. Выдели фундаментальные идеи для "вечнозеленых заметок"
16. Рекомендуй формат для практических примеров и шаблонов кода
17. Предложи методику регулярного обзора заметок

Структурируй по разделам с примерами кода и диаграммами.
```

## Применение

- Изучение новых технологий
- Создание базы знаний
- Структурирование обучения
- Подготовка к сертификациям
